# Dashboard de Ventas 2025

Proyecto de visualización y análisis de datos desarrollado con Google Data Studio para evaluar ventas por vendedor, comportamiento mensual y participación porcentual dentro de una organización comercial.

## Tecnologías

* Google Data Studio
* Canva

## Contenido

* Dashboard interactivo
* Análisis de ventas por vendedor
* Evolución mensual de ventas
* Ranking y participación porcentual
* Indicadores visuales de desempeño
* Presentación ejecutiva en PDF

## Objetivo del Proyecto

Transformar datos de ventas en información visual útil para la toma de decisiones mediante dashboards interactivos y reportes ejecutivos.
